FAQ

Frequently asked questions

Is AI Web Reader free?

Yes, AI Web Reader is completely free to use. Install it from the Chrome Web Store and start chatting with any webpage instantly.

How does it work under the hood?

AI Web Reader uses RAG (Retrieval-Augmented Generation). When you analyze a page, the content is split into chunks and embedded locally using TF-IDF. When you ask a question, the most relevant chunks are retrieved via cosine similarity and sent to the AI for an accurate, grounded response.

Is my data safe?

Absolutely. Page content is processed locally for indexing. Only the small, relevant text chunks needed to answer your question are sent to the AI — never the full page content.

Does it work on any website?

It works on virtually any public webpage — articles, blogs, documentation, research papers, and more. It automatically filters out navigation, ads, and boilerplate to focus on the main content.

Does it support single-page applications (SPAs)?

Yes! AI Web Reader automatically detects navigation and content changes in SPAs. It re-indexes content in the background so your chat stays up to date as you navigate.

Can I use it on multiple tabs?

Yes. Each tab maintains its own context and chat history. You can analyze and chat on multiple pages simultaneously.
